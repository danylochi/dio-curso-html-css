# Projeto DIO - Curso de HTML e CSS

## Descrição
Este repositório reúne um projeto prático desenvolvido durante o curso de **HTML e CSS** da [DIO](https://www.dio.me/). A proposta foi construir uma página estática de apresentação pessoal para praticar estruturação de conteúdo com HTML e estilização com CSS.

## O que foi construído
A página principal apresenta:
- Cabeçalho com imagem e nome;
- Seção de posts com um artigo de exemplo;
- Links de contato no rodapé (e-mail, LinkedIn e GitHub).

A estrutura foi organizada com elementos semânticos como `header`, `section`, `article` e `footer`.

## Tecnologias utilizadas
- HTML5
- CSS3

## Conceitos praticados
- Estrutura base de um documento HTML;
- Organização de conteúdo com tags semânticas;
- Uso de seletores CSS por elemento, classe e id;
- Estilização de blocos com `background`, `border`, `margin` e `padding`;
- Inserção de imagens e links externos.

## Como visualizar localmente
Abra o arquivo `digitalino.html` em qualquer navegador.
